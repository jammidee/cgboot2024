# Template For a Module
- Created by Jammi Dee
- Date: 03/21/2026


This example shows how to create a feature in a module format.

---
## Setup

The design is to make sure that all the classes resides on a single folder inside the **modules** subfolder and that all the views also resides on the **view/modules** subfolder also.

Only framework calls and templates are reference outside the modulefolder.
